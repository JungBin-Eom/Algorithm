# Algorithm
이 repository는 Ricky와 YUMYUMNYANG의 겨울방학 알고리즘 스터디 코드 저장소입니다.
  
---
  
### Rule
1. 매주 일요일 저녁 다음주에 풀 문제를 선정한다.
2. 한 주당 하나의 문제를 해결한다.
3. 자신이 해결한 방법(알고리즘, 구현)과 어려웠던 점을 기록한다.
4. 문제를 해결하지 못하거나 기록을 하지 않았을 시 스타벅스 아메리카노 기프티콘을 상대방에게 준다.
